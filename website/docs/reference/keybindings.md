# 键盘快捷键

Elevim 提供了一系列高效的快捷键来提升你的编辑体验。

## 文件操作

| 命令 | Windows/Linux | 说明 |
| :--- | :--- | :--- |
| **新建文件** | `Ctrl` + `N` | 创建一个新的空标签页 |
| **打开文件** | `Ctrl` + `O` | 打开文件选择对话框 |
| **打开文件夹** | `Ctrl` + `Shift` + `O` | 打开项目文件夹 |
| **保存** | `Ctrl` + `S` | 保存当前文件 |
| **另存为** | `Ctrl` + `Shift` + `S` | 将当前文件另存为新文件 |
| **退出** | `Ctrl` + `Q` | 退出应用 |

## 视图与导航

| 命令 | Windows/Linux | 说明 |
| :--- | :--- | :--- |
| **命令面板** | `Ctrl` + `Shift` + `P` | 打开全局命令搜索面板 |
| **切换终端** | `Ctrl` + `` ` `` | 显示或隐藏底部集成终端 |
| **源代码管理** | `Ctrl` + `Shift` + `G` | 切换 Git 面板 |
| **搜索** | `Ctrl` + `Shift` + `F` | 切换全局搜索面板 |
| **拆分编辑器** | `Ctrl` + `\` | 将编辑器拆分为两栏 |
| **禅模式** | `Ctrl` + `Alt` + `M` | 切换全屏沉浸式编辑模式 |
| **下一个编辑器** | `Ctrl` + `Tab` | 切换到右侧标签页 |
| **上一个编辑器** | `Ctrl` + `Shift` + `Tab` | 切换到左侧标签页 |
