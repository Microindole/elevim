
# 核心功能

## 编辑器

Elevim 的核心基于 **CodeMirror 6**，提供了高性能的文本编辑体验。

* **语法高亮**: 支持多种主流编程语言。
* **智能补全**: 基于 LSP (Language Server Protocol) 的代码提示（开发中）。
* **多光标支持**: 按住 `Alt` 点击可添加多个光标。

## 界面布局

Elevim 采用了现代 IDE 的标准布局：

1.  **活动栏 (Activity Bar)**: 位于最左侧，用于在资源管理器、Git、搜索等视图间切换。
2.  **侧边栏 (Sidebar)**: 显示当前活动视图的详细内容（如文件树）。
3.  **编辑区 (Editor Groups)**: 支持垂直/水平拆分，支持拖拽调整大小。
4.  **面板 (Panel)**: 底部区域，包含集成终端和状态栏。

## 禅模式 (Zen Mode)

通过快捷键 `Ctrl + Alt + M` 触发。
在此模式下，所有侧边栏、状态栏和顶部标题栏都会隐藏，编辑器将占据整个窗口，让你专注于代码本身。
