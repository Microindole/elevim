# 知识库管理 (Knowledge Base)

Elevim 不仅仅是一个代码编辑器，它还内置了先进的“双向链接”系统，帮助你将零散的 Markdown 笔记编织成个人的知识网络。

## 双向链接 (Wiki Links)

你可以使用 `[[ ]]` 语法在文件之间建立连接，无需关心文件的具体路径。

### 基础链接
输入 `[[` 即可触发自动补全，选择目标文件后：

这是一条指向 [[MyNote]] 的链接。
点击该链接，编辑器会自动跳转到 MyNote.md。如果文件不存在，Elevim 会询问你是否创建它。

使用别名
你可以为链接设置一个“显示文本”，使其在阅读时更自然：

Markdown

[[2023-10-24-meeting-notes|周一会议记录]]
阅读模式：编辑器仅显示“<u>周一会议记录</u>”。

编辑模式：将光标移动到链接上，它会自动展开为原始语法 [[...|...]] 供你修改。

悬停预览 (Hover Preview)
无需打开文件，只需将鼠标悬停在任何 [[链接]] 上，即可在一个浮层中快速预览目标文件的内容摘要。这在查阅定义或回顾笔记时非常有用。

反向链接 (Backlinks)
想知道当前文件被哪些其他文件引用了吗？

打开任意文件。

点击活动栏左侧的 References (引用) 图标（链接形状）。

侧边栏将显示所有引用了当前文件的源文件列表，并展示引用所在的行内容。

通过反向链接，你可以轻松发现知识点之间的潜在关联，构建网状的知识结构。
