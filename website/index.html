# Welcome to Zen Mode

当你进入 Zen Mode，世界安静了。
所有的 UI 都会隐去，只留下文字。

功能特性：
- 专注于当前行的聚焦模式
- 永不触底的打字机滚动
- 丝滑的界面过渡动画
- 极速启动，轻量资源占用

Ready to create?
